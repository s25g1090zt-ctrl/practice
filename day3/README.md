# 商品管理システム

## 機能
商品名とその価格を入力することで，合計金額と平均価格を出します．

## 使い方
1. `gcc product.c`
2. `./a.out`

## 学んだこと

- 構造体 
 - 配列は，同じ型( int なら int だけ)しか入らないが，構造体は，いろんな型が入れる．
 - struct 構造体の名前　変数の名前
 - 構造体の変数の後に .aaa をつけ，何の箱なのか示す．（　list[0].price = 100; 0番目の商品の「価格」に100を入れる）
- 関数
 - 関数の引数の順番は，最小に定義した順番にする．
- 書式指定子
 - int -> %d
 - double -> %f ( .1 は少数点第一まで)
 - char -> %s

- github
 - ブランチ運用: feature ブランチで新機能（最高価格検索）を開発し，動作確認後に main へマージする流れを実践した．
